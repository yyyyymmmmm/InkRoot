## 📋 版本概述

v0.27.0 是 Memos 的突破性版本，引入了**实时通信（SSE）**、**语音笔记**、**AI 转录**、**MCP 集成**等重大功能。这是目前功能最丰富的版本。

### 🎯 核心新功能
- 🔴 **Server-Sent Events (SSE)**：实时更新推送
- 🎤 **语音笔记录制**：编辑器内语音输入
- 🤖 **AI 音频转录**：Gemini/BYOK 转录支持
- 🔌 **MCP 服务器**：Model Context Protocol 集成
- 🔗 **备忘录分享链接**：可设置过期时间的分享
- 💬 **备忘录提及**：@username 功能
- 📸 **Live Photo 支持**：动态照片和运动照片

### ⚠️ Breaking Changes
- **移除公共 ActivityService**：迁移到用户通知 API
- **用户资源名称改用用户名**：从 `users/1` → `users/alice`
- **身份提供商资源名称使用 UID**：替换自增 ID
- **移除 disallow_public_visibility 设置**
- **默认附件存储改为本地文件系统**

## 📡 Server-Sent Events (SSE) - 新增

### 实时更新连接

**建立 SSE 连接**
```http
GET /api/v1/sse
Authorization: Bearer <access_token>
Accept: text/event-stream
```

**事件格式**
```
event: memo.created
data: {"id": 100, "content": "New memo", "createdTs": 1234567890}

event: memo.updated
data: {"id": 100, "content": "Updated memo", "updatedTs": 1234567891}

event: memo.deleted
data: {"id": 100}

event: comment.created
data: {"id": 1, "memoId": 100, "content": "Great post!", "createdTs": 1234567892}

event: reaction.added
data: {"memoId": 100, "reactionType": "👍", "userId": 1}

event: connection.status
data: {"status": "connected", "timestamp": 1234567890}
```

### 支持的事件类型

| 事件类型 | 触发条件 | 数据内容 |
|---------|---------|---------|
| `memo.created` | 新建备忘录 | 完整备忘录对象 |
| `memo.updated` | 更新备忘录/附件/关系 | 完整备忘录对象 |
| `memo.deleted` | 删除备忘录 | 备忘录 ID |
| `comment.created` | 新增评论 | 评论对象 |
| `comment.updated` | 更新评论 | 评论对象 |
| `comment.deleted` | 删除评论 | 评论 ID |
| `reaction.added` | 添加反应 | 反应对象 |
| `reaction.removed` | 移除反应 | 反应对象 |
| `connection.status` | 连接状态变化 | 状态信息 |

### 前端集成示例

```javascript
class MemosSSEClient {
  constructor(baseUrl, accessToken) {
    this.baseUrl = baseUrl;
    this.accessToken = accessToken;
    this.eventSource = null;
    this.listeners = new Map();
  }

  connect() {
    this.eventSource = new EventSource(
      `${this.baseUrl}/api/v1/sse`,
      {
        headers: {
          'Authorization': `Bearer ${this.accessToken}`
        }
      }
    );

    // 监听备忘录创建
    this.eventSource.addEventListener('memo.created', (e) => {
      const memo = JSON.parse(e.data);
      this.emit('memo.created', memo);
    });

    // 监听备忘录更新
    this.eventSource.addEventListener('memo.updated', (e) => {
      const memo = JSON.parse(e.data);
      this.emit('memo.updated', memo);
    });

    // 监听备忘录删除
    this.eventSource.addEventListener('memo.deleted', (e) => {
      const data = JSON.parse(e.data);
      this.emit('memo.deleted', data.id);
    });

    // 监听评论
    this.eventSource.addEventListener('comment.created', (e) => {
      const comment = JSON.parse(e.data);
      this.emit('comment.created', comment);
    });

    // 连接状态
    this.eventSource.addEventListener('connection.status', (e) => {
      const status = JSON.parse(e.data);
      this.emit('connection', status);
    });

    // 错误处理
    this.eventSource.onerror = (error) => {
      console.error('SSE connection error:', error);
      this.emit('error', error);
    };
  }

  on(event, callback) {
    if (!this.listeners.has(event)) {
      this.listeners.set(event, []);
    }
    this.listeners.get(event).push(callback);
  }

  emit(event, data) {
    const callbacks = this.listeners.get(event);
    if (callbacks) {
      callbacks.forEach(cb => cb(data));
    }
  }

  disconnect() {
    if (this.eventSource) {
      this.eventSource.close();
      this.eventSource = null;
    }
  }
}

// 使用示例
const sseClient = new MemosSSEClient('https://memos.example.com', accessToken);

sseClient.on('memo.created', (memo) => {
  console.log('New memo:', memo);
  // 更新 UI
});

sseClient.on('memo.updated', (memo) => {
  console.log('Updated memo:', memo);
  // 刷新备忘录列表
});

sseClient.on('connection', (status) => {
  console.log('Connection status:', status);
  // 显示连接指示器
});

sseClient.connect();
```

```http

Authorization: Bearer <access_token>
```

**响应**
```json
{
  "transcription": "转录的文本内容",
  "language": "zh-CN",
  "confidence": 0.95,
  "duration": 15.5
}
```

## 🔌 MCP (Model Context Protocol) - 新增

### MCP 端点

v0.27.0 内置 MCP 服务器，供 AI 客户端访问。

**端点**: `/mcp`
**协议**: Streamable HTTP
**认证**: Personal Access Token (PAT)

### 支持的 MCP 工具

| 工具名称 | 功能 | 参数 |
|---------|------|------|
| `memo.list` | 列出备忘录 | `limit`, `offset`, `tag`, `visibility` |
| `memo.create` | 创建备忘录 | `content`, `visibility` |
| `memo.get` | 获取备忘录 | `id` |
| `memo.update` | 更新备忘录 | `id`, `content`, `visibility` |
| `memo.delete` | 删除备忘录 | `id` |
| `comment.list` | 列出评论 | `memoId` |
| `comment.create` | 创建评论 | `memoId`, `content` |
| `tag.list` | 列出标签 | - |
| `attachment.upload` | 上传附件 | `file` |
| `relation.create` | 创建关系 | `memoId`, `relatedMemoId` |

### MCP 客户端示例

```javascript
// Claude Desktop MCP 配置
{
  "memos": {
    "command": "curl",
    "args": [
      "-X", "POST",
      "-H", "Authorization: Bearer memos_pat_xxxxxx",
      "-H", "Content-Type: application/json",
      "https://memos.example.com/mcp"
    ]
  }
}
```

```json
}
```

匹配所有以 `work` 开头的标签：`#work`, `#work-project`, `#work-meeting`

## 💡 最佳实践

### 1. SSE 连接管理

```javascript
class SSEManager {
  constructor() {
    this.client = null;
    this.reconnectDelay = 1000;
    this.maxReconnectDelay = 30000;
  }

  connect(baseUrl, token) {
    this.client = new MemosSSEClient(baseUrl, token);

    this.client.on('error', () => {
      // 指数退避重连
      setTimeout(() => {
        this.connect(baseUrl, token);
        this.reconnectDelay = Math.min(
          this.reconnectDelay * 2,
          this.maxReconnectDelay
        );
      }, this.reconnectDelay);
    });

    this.client.on('connection', (status) => {
      if (status.status === 'connected') {
        this.reconnectDelay = 1000; // 重置延迟
      }
    });

    this.client.connect();
  }
}
```

### 2. 语音笔记集成

```javascript
async function recordVoiceNote() {
  const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
  const mediaRecorder = new MediaRecorder(stream);
  const chunks = [];

  mediaRecorder.ondataavailable = (e) => chunks.push(e.data);

  mediaRecorder.onstop = async () => {
    const blob = new Blob(chunks, { type: 'audio/webm' });
    const formData = new FormData();
    formData.append('audio', blob);
    formData.append('transcribe', 'true');

    const response = await fetch('/api/v1/memo/voice-note', {
      method: 'POST',
      headers: { 'Authorization': `Bearer ${token}` },
      body: formData
    });

    const { resourceId, transcription } = await response.json();
    console.log('Transcription:', transcription);
  };

  mediaRecorder.start();
  setTimeout(() => mediaRecorder.stop(), 5000); // 录制 5 秒
}
```

### 3. MCP 集成

```python
# Python MCP Client
import requests

class MemosMCPClient:
    def __init__(self, base_url, pat):
        self.base_url = base_url
        self.pat = pat

    def call_tool(self, tool_name, params):
        response = requests.post(
            f"{self.base_url}/mcp",
            headers={
                "Authorization": f"Bearer {self.pat}",
                "Content-Type": "application/json"
            },
            json={
                "tool": tool_name,
                "params": params
            }
        )
        return response.json()

    def list_memos(self, tag=None):
        return self.call_tool("memo.list", {"tag": tag})

    def create_memo(self, content):
        return self.call_tool("memo.create", {"content": content})

# 使用
client = MemosMCPClient("https://memos.example.com", "memos_pat_xxx")
memos = client.list_memos(tag="work")
```
