## 📋 版本概述

v0.24.0 是 Memos 的重大功能更新版本，引入了**快捷方式系统**，允许用户创建可重用的过滤器。该版本包含数据库和 API 变更，建议升级前备份数据。

### 🎯 核心新功能
- ✨ **快捷方式系统**：基于条件快速过滤备忘录
- ✨ 支持比较、集合、字符串和逻辑运算符
- ✨ 快捷方式可固定、重命名、删除
- ✅ 继承 v0.21.0 所有功能
- ✅ 改进的备忘录过滤引擎

### ⚠️ Breaking Changes
- 数据库架构变更（需要迁移）
- 新增快捷方式相关表

## 🔖 快捷方式 API（新增）

### 快捷方式数据结构

```typescript
interface Shortcut {
  id: number;
  creatorId: number;
  name: string;
  filter: string; // CEL (Common Expression Language) 表达式
  isPinned: boolean;
  createdTs: number;
  updatedTs: number;
}
```

### 创建快捷方式

**请求**
```http
POST /api/shortcut
Content-Type: application/json
Cookie: session_token=xxx

{
  "name": "Work Notes",
  "filter": "tag == 'work' && visibility == 'PRIVATE'",
  "isPinned": false
}
```

**过滤器表达式（CEL 语法）**

支持的字段：
- `content`: 备忘录内容
- `tag`: 标签
- `visibility`: 可见性
- `pinned`: 是否置顶
- `createdTs`: 创建时间
- `updatedTs`: 更新时间

支持的运算符：
- **比较**: `==`, `!=`, `<`, `<=`, `>`, `>=`
- **逻辑**: `&&` (AND), `||` (OR), `!` (NOT)
- **集合**: `in`, `has`
- **字符串**: `contains`, `startsWith`, `endsWith`

**过滤器示例**

```javascript
// 包含特定标签
"tag == 'work'"

// 多个标签（AND）
"tag == 'work' && tag == 'important'"

// 多个标签（OR）
"tag == 'work' || tag == 'personal'"

// 内容包含关键词
"content.contains('meeting')"

// 时间范围（最近 7 天）
"createdTs >= now() - duration('168h')"

// 组合条件
"(tag == 'work' || tag == 'project') && visibility == 'PRIVATE'"

// 置顶的公开备忘录
"pinned == true && visibility == 'PUBLIC'"

// 不包含特定标签
"!(tag == 'archive')"
```

**响应**
```json
{
  "id": 1,
  "creatorId": 1,
  "name": "Work Notes",
  "filter": "tag == 'work' && visibility == 'PRIVATE'",
  "isPinned": false,
  "createdTs": 1234567890,
  "updatedTs": 1234567890
}
```

## 🔍 高级过滤与搜索（增强）

### CEL 表达式详解

v0.24.0 使用 CEL (Common Expression Language) 作为过滤器引擎。

#### 基础语法

**字段访问**
```javascript
tag         // 访问 tag 字段
content     // 访问 content 字段
visibility  // 访问 visibility 字段
```

**字符串比较**
```javascript
tag == 'work'              // 等于
tag != 'archive'           // 不等于
visibility == 'PUBLIC'     // 枚举值比较
```

**数值比较**
```javascript
createdTs > 1234567890     // 大于
updatedTs >= 1234567890    // 大于等于
createdTs < 1234567900     // 小于
```

**布尔比较**
```javascript
pinned == true             // 置顶
pinned == false            // 未置顶
```

**逻辑运算**
```javascript
tag == 'work' && pinned == true          // AND
tag == 'work' || tag == 'personal'       // OR
!(tag == 'archive')                      // NOT
```

**字符串函数**
```javascript
content.contains('meeting')              // 内容包含
content.startsWith('TODO')               // 内容以...开头
content.endsWith('done')                 // 内容以...结尾
```

**集合运算**
```javascript
tag in ['work', 'personal', 'important'] // 标签在集合中
'work' in tags                           // tags 包含 'work'
```

**时间函数**
```javascript
createdTs >= now() - duration('7d')      // 最近 7 天
createdTs >= now() - duration('168h')    // 最近 168 小时（7天）
createdTs >= now() - duration('1h30m')   // 最近 1 小时 30 分钟
```

#### 复杂表达式示例

**工作相关的重要备忘录**
```javascript
(tag == 'work' || tag == 'project') && tag == 'important'
```

**最近一周的公开备忘录**
```javascript
visibility == 'PUBLIC' && createdTs >= now() - duration('7d')
```

**包含待办事项的未完成备忘录**
```javascript
content.contains('TODO') && !content.contains('DONE')
```

**多标签过滤**
```javascript
tag in ['work', 'personal'] && visibility == 'PRIVATE'
```

**排除归档的所有备忘录**
```javascript
!(tag == 'archive' || tag == 'deleted')
```

## 🎨 前端集成示例

### 快捷方式管理器

```javascript
class ShortcutManager {
  constructor(apiBase, sessionToken) {
    this.apiBase = apiBase;
    this.headers = {
      'Content-Type': 'application/json',
      'Cookie': `session_token=${sessionToken}`
    };
  }

  // 创建快捷方式
  async create(name, filter, isPinned = false) {
    const response = await fetch(`${this.apiBase}/shortcut`, {
      method: 'POST',
      headers: this.headers,
      credentials: 'include',
      body: JSON.stringify({ name, filter, isPinned })
    });
    return response.json();
  }

  // 获取所有快捷方式
  async list() {
    const response = await fetch(`${this.apiBase}/shortcut`, {
      headers: this.headers,
      credentials: 'include'
    });
    return response.json();
  }

  // 更新快捷方式
  async update(id, updates) {
    const response = await fetch(`${this.apiBase}/shortcut/${id}`, {
      method: 'PATCH',
      headers: this.headers,
      credentials: 'include',
      body: JSON.stringify(updates)
    });
    return response.json();
  }

  // 删除快捷方式
  async delete(id) {
    const response = await fetch(`${this.apiBase}/shortcut/${id}`, {
      method: 'DELETE',
      headers: this.headers,
      credentials: 'include'
    });
    return response.json();
  }

  // 应用快捷方式过滤
  async applyFilter(shortcutId) {
    const response = await fetch(`${this.apiBase}/memo?shortcutId=${shortcutId}`, {
      headers: this.headers,
      credentials: 'include'
    });
    return response.json();
  }
}

// 使用示例
const manager = new ShortcutManager('/api', 'your_session_token');

// 创建"工作笔记"快捷方式
const shortcut = await manager.create(
  'Work Notes',
  "tag == 'work' && visibility == 'PRIVATE'",
  true
);

// 应用过滤器
const filteredMemos = await manager.applyFilter(shortcut.id);
console.log('Filtered memos:', filteredMemos);
```

### CEL 过滤器构建器

```javascript
class FilterBuilder {
  constructor() {
    this.conditions = [];
  }

  // 添加标签条件
  tag(tag) {
    this.conditions.push(`tag == '${tag}'`);
    return this;
  }

  // 添加可见性条件
  visibility(visibility) {
    this.conditions.push(`visibility == '${visibility}'`);
    return this;
  }

  // 添加内容包含条件
  contains(keyword) {
    this.conditions.push(`content.contains('${keyword}')`);
    return this;
  }

  // 添加时间范围条件
  recent(days) {
    this.conditions.push(`createdTs >= now() - duration('${days}d')`);
    return this;
  }

  // 添加置顶条件
  pinned(isPinned = true) {
    this.conditions.push(`pinned == ${isPinned}`);
    return this;
  }

  // 构建 AND 表达式
  buildAnd() {
    return this.conditions.join(' && ');
  }

  // 构建 OR 表达式
  buildOr() {
    return this.conditions.join(' || ');
  }

  // 重置
  reset() {
    this.conditions = [];
    return this;
  }
}

// 使用示例
const builder = new FilterBuilder();

// 构建过滤器：最近 7 天的工作笔记（私密）
const filter = builder
  .tag('work')
  .visibility('PRIVATE')
  .recent(7)
  .buildAnd();

console.log(filter);
// 输出: tag == 'work' && visibility == 'PRIVATE' && createdTs >= now() - duration('7d')

// 创建快捷方式
await manager.create('Recent Work Notes', filter, true);
```

### 4. 错误处理

```javascript
async function createShortcut(name, filter) {
  try {
    // 验证过滤器语法
    if (!isValidCEL(filter)) {
      throw new Error('Invalid filter syntax');
    }

    const shortcut = await manager.create(name, filter);

    // 测试过滤器是否返回结果
    const memos = await manager.applyFilter(shortcut.id);

    if (memos.length === 0) {
      console.warn('Filter returned no results');
    }

    return shortcut;
  } catch (error) {
    if (error.status === 400) {
      console.error('Invalid filter expression:', filter);
    }
    throw error;
  }
}

// 简单的 CEL 验证
function isValidCEL(expression) {
  // 检查括号匹配
  const stack = [];
  for (const char of expression) {
    if (char === '(') stack.push(char);
    else if (char === ')') {
      if (stack.length === 0) return false;
      stack.pop();
    }
  }
  return stack.length === 0;
}
```
