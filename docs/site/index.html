InkRoot 墨鸣笔记 · v1.1.13

静待沉淀，蓄势而鸣

一个本地优先的笔记客户端。你可以离线记录，也可以连接官方服务器或自部署 Memos；图片、标签、分享、备份和小组件都围绕“少打扰地写下来”设计。

下载最新版 GitHub 仓库
1.1.13 已发布：修复 Android 分享闪退，随机回顾小组件改为显示真实笔记快照，并继续校验 Android 正式签名。

给真正写笔记的人

InkRoot 不把记录做成复杂系统。默认先保存到本机，需要同步时再接入你选择的服务。

本地优先

没有网络也能记录。笔记、图片和设置优先保存在设备上，适合临时灵感、读书摘录和日常备忘。

连接 Memos

支持官方服务器，也支持自部署 Memos。当前兼容重点覆盖 0.21.0 之后的登录、同步、资源和引用接口差异。

备份自己掌握

WebDAV 可备份笔记和图片附件，适合把资料放到自己的云盘或 NAS。导入导出也保留常规迁移路径。

记录入口更顺手

记录不应该被打开 App 的流程打断。InkRoot 正在把常用入口补齐：系统分享、桌面小组件、离线发布、图片和文件附件。

系统分享

Android 支持从浏览器、阅读器、相册、文件等入口分享到 InkRoot。iOS 分享扩展已加入构建链路。

桌面小组件

记录小组件和随机回顾小组件分开。随机回顾使用应用内真实笔记快照，不再显示占位文案。

离线发布

网络不稳定时先保存，恢复连接后再同步。适合自部署服务器不在同一网络的场景。

AI 是可选能力，不是前提

摘要、续写、标签推荐、点评和洞察可以使用你自己配置的模型。没有配置 AI 时，相关笔记和本地点评仍尽量提供可用结果。

用户自填模型

模型更新太快，InkRoot 不把选择写死。你可以填写 API 地址、模型名和 Key。

相关笔记保留

相关笔记是独立能力，不会因为洞察或点评改造而消失。它更适合在详情页连接上下文。

输出少套话

AI 功能的目标是给出和笔记内容相关的具体反馈，不做万能模板式评价。

当前支持的平台

GitHub Release 会自动构建 Android、iOS unsigned、macOS、Windows 和 Linux 产物。Android 正式包会校验固定发布证书，确保覆盖安装链路一致。
