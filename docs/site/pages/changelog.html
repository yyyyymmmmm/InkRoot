更新日志

只保留用户需要知道的变化。完整提交记录和构建产物请查看 GitHub Release。

v1.1.13

2026-06-22
  • 修复 Android 从浏览器、阅读器分享文本或网页到 InkRoot 时可能闪退的问题。
  • 修复 Android 分享入口把网页 URL ClipData 误判为附件的问题。
  • 修复 iOS/Android 随机回顾小组件明明有笔记却显示占位文案的问题。
  • 随机回顾小组件改为按笔记显示时间筛选，避免同步或恢复后被更新时间污染。
  • 补齐 iOS 主 App、小组件和分享扩展的 Development Team 配置，便于 Xcode 真机调试。
  • 随机回顾小组件设置变更后立即刷新快照。
  • 分享载荷解析增强容错，避免平台通道异常类型导致崩溃。

v1.1.12

2026-06-22
  • 修复 iOS 分享扩展加入后 unsigned IPA 构建继承 Runner Flutter 链接配置导致发布流程失败的问题。

v1.1.11

2026-06-22
  • 新增 iOS 系统分享扩展，支持从浏览器、阅读器、相册和文件 App 分享文本、网页、图片与文件到 InkRoot。
  • 优化 Android 系统分享入口，覆盖浏览器、阅读器、相册和文件管理器等来源。
  • 增强 Memos 0.21.0 至 0.29.1 的服务器识别、登录 token 字段和当前用户接口回退能力。
  • 修复部分 Android App 分享到 InkRoot 后内容为空白的问题。

v1.1.10

2026-06-22
  • 新增 Android 和 iOS 桌面小组件基础能力，记录小组件和随机回顾小组件分离。
  • 新增随机回顾小组件内容选择与刷新配置。
  • 新增讯飞语音识别自定义配置入口。
  • 优化知识图谱视觉、缩放、节点点击和大数据量渲染性能。
  • 优化桌面端全局缩放、Markdown 标题/引用/链接/下划线渲染和中文标签解析。

v1.1.8 - v1.1.9

2026-06-16
  • 修复待办点击完成、下划线和加粗主页渲染、搜索结果空白和相关笔记入口等问题。
  • 优化 AI 点评、摘要、续写、标签推荐和洞察输出，减少模板化表达。
  • 优化关于页、帮助页、设置页和法律文档内容。
  • 修复发布流程中 iOS 依赖下载超时导致的构建中断。

v1.1.0 - v1.1.7

2026-06-14 至 2026-06-15
  • 新增统一维护 CLI：dart tool/inkroot.dart。
  • GitHub Actions 支持 Android、iOS 模拟器、macOS、Windows、Linux 验证与构建。
  • 新增账号与数据删除入口、隐私政策、用户协议和 iOS 隐私清单。
  • 固定官方云验证配置，Android Release 强制签名密钥和证书指纹校验。
  • 修复 Memos 登录兼容、Windows 默认图标、Android release 构建等问题。
