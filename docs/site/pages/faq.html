FAQ

常见问题

这里整理 InkRoot 的安装、记录、同步、备份、迁移和隐私问题。没有覆盖到的情况，请带上平台、InkRoot 版本、Memos 版本和错误提示提交 Issue。

安装与平台

应该下载哪个包？

Android 用户下载 APK；Google Play 或商店发布流程使用 AAB。iOS 当前提供 unsigned IPA，需要签名或侧载。macOS、Windows、Linux 下载对应压缩包并解压运行。

iOS unsigned IPA 是正式版吗？

不是 App Store 正式包。unsigned IPA 主要用于测试和签名流程。正式上架前，还需要 Apple Developer 账号、证书、描述文件、TestFlight 测试和 App Store Review。

桌面端和移动端的数据会自动互通吗？

本地模式下数据只保存在当前设备。需要跨设备同步时，可以登录官方服务器或自部署 Memos；需要备份和迁移时，可以使用导出或 WebDAV 备份。

记录与编辑

InkRoot 是 Markdown 软件吗？

InkRoot 使用兼容 Memos 的 Markdown 作为存储格式，但编辑体验尽量接近普通富文本：加粗、下划线、链接、待办等操作通过工具栏完成，用户不必先记语法。

可以离线记录吗？

可以。InkRoot 默认本地优先，网络不可用时仍可记录。连接恢复后，再按你选择的服务器或同步方式处理。

支持待办、链接、标签和图片吗？

支持。笔记内容可以包含待办、链接、图片和标签；标签支持多级路径；图片支持上传、预览、保存和多图浏览。

Memos 与同步

自部署 Memos 只用 IP + 端口可以吗？

可以。服务器地址可以是 http://192.168.x.x:5230 这类内网地址，也可以是域名。关键是手机或电脑必须能访问这个地址；如果在外网使用内网 IP，通常无法连接。

支持哪些 Memos 版本？

当前兼容重点覆盖 Memos 0.21.0 之后的常见版本。不同版本在登录、当前用户、资源、附件和引用接口上有差异，InkRoot 会自动做回退处理。遇到问题请提供 Memos 版本号。

官方服务器和自部署 Memos 有什么区别？

官方服务器适合希望快速开始同步的用户；自部署 Memos 适合已经有自己服务器、NAS 或私有化部署需求的用户。无论选择哪种方式，都建议定期备份重要数据。

为什么登录成功后仍可能同步失败？

常见原因包括服务器地址无法被当前网络访问、Memos 版本接口差异、反向代理路径配置、Token 或 Cookie 鉴权差异、附件资源接口不可用。请先确认浏览器能访问同一地址，再提交具体版本和错误提示。

备份、恢复与迁移

WebDAV 会备份图片吗？

可以。WebDAV 配置里有图片附件备份选项。图片较多时，首次备份会更慢，也会占用更多空间。

WebDAV 是实时同步吗？

不是。WebDAV 在 InkRoot 中主要用于备份和恢复，不等同于多人协作或实时双向同步。恢复前建议先导出当前数据，避免误覆盖。

可以从其他笔记工具迁移吗？

当前提供 Flomo、微信读书和本地备份等导入路径。不同工具导出的格式不完全一致，导入前建议保留原始备份文件。

分享、图片与小组件

可以从哪些入口分享到 InkRoot？

Android 支持从浏览器、阅读器、系统选中文本、相册和文件管理器等入口分享文本、网页、图片和文件。iOS 分享扩展已进入构建链路，具体入口以系统分享面板显示为准。

从浏览器或阅读器分享进来为空白怎么办？

请升级到 v1.1.13 或更高版本。该版本修复了 Android 浏览器 URL 被误判为附件、分享载荷类型异常导致闪退或空白的问题。如果仍复现，请提供来源 App、分享内容类型和系统版本。

随机回顾小组件为什么显示空？

先打开 InkRoot，让应用生成一次小组件快照。v1.1.13 已修复有笔记却显示占位文案的问题；如果仍为空，检查随机回顾范围和标签筛选是否过窄。

图片会怎么保存和展示？

图片可以作为笔记附件上传和预览，也可以在图片查看器中放大、浏览多图和保存。自部署 Memos、离线发布和 WebDAV 备份场景下，图片能否远端显示取决于服务器资源接口和备份配置。

整理与 AI

标签、搜索、相关笔记和知识图谱分别解决什么问题？

标签用于主动分类，搜索用于快速找内容，相关笔记用于发现上下文，知识图谱用于观察内容关系。它们是互补能力，不是互相替代。

AI 功能必须配置吗？

不必须。记录、同步、备份、标签、搜索和相关笔记可以在不配置 AI 的情况下使用。AI 功能需要用户自己填写 API 地址、模型名和 Key。

为什么模型选择是输入框？

AI 模型更新很快，固定下拉列表容易过期。InkRoot 让用户直接填写模型名，便于接入自己使用的 OpenAI 兼容接口或其他服务。

隐私与账号

如何删除账号和数据？

本机数据可在应用内删除。官方服务器账号可通过 账号与数据删除页面 提交申请。自部署 Memos、WebDAV 或第三方 AI 服务中的数据，需要到对应服务里删除。

InkRoot 会读取我的私人笔记做广告分析吗？

不会。隐私政策中明确说明不会出售个人信息，也不会读取或分析私人笔记用于广告画像。只有当你主动登录、同步、备份、启用 AI 或提交反馈时，相关数据才会发送到对应服务。
