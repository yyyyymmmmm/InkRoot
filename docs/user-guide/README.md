## 🎯 What is InkRoot?

InkRoot is a powerful yet simple note-taking app that helps you:

- 📝 **Capture Ideas Quickly** - Write notes with full Markdown support
- ✅ **Manage Tasks** - Interactive todo lists
- 🎤 **Voice Input** - Convert speech to text
- ☁️ **Sync Everywhere** - Optional cloud sync or local-only mode
- 🤖 **AI-Powered** - Intelligent note enhancement
- 🕸️ **Build Knowledge** - Connect notes with knowledge graph
- 🔒 **Privacy First** - Your data, your control

---

## 🚀 Quick Start

### 1. Choose Your Mode

**Local Mode** (No server needed)
- All data stays on your device
- No account required
- Perfect for personal use

**Cloud Sync Mode** (Optional)
- Sync across devices
- Requires Memos server
- Great for team collaboration

### 2. Create Your First Note

1. Tap the **+** button
2. Write your note using Markdown
3. Add tags with `#tagname`
4. Save automatically

### 3. Explore Features

- Try **voice input** 🎤
- Create **todo lists** ✅
- Upload **images** 📷
- Set **reminders** ⏰

## 💡 Tips & Tricks

### Writing Tips
- Use `#tags` for quick organization
- Create templates with common structures
- Use voice input for quick capture
- Add images for visual context

### Organization Tips
- Pin important notes to top
- Use consistent tagging system
- Review notes regularly
- Create note references for related content

### Productivity Tips
- Set reminders for actionable items
- Use todo lists for task tracking
- Use AI assistant for brainstorming
- Export regularly for backup

### Advanced Tips
- Build knowledge networks with references
- Use knowledge graph to discover connections
- Customize fonts for comfortable reading
- Enable dark mode for night writing
